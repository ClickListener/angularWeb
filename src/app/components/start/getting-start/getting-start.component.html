Developer Registration Application Instruction

Step 1: Open your browser (IE8 and above, Firefox, Chrome, Safari)

Step 2: Go to https://dev.ihealthlabs.com/

Step 3: Create developer iHealth account

Step 4: Complete company info

Go to Enterprise Development -> Apply

Complete company info and submit

Step 5: Add your application

Go to Enterprise Development -> App Management -> Add New App

Complete app info and submit

Step 6: Application approved

Go to Enterprise Development -> App Management

Step 7: Download SDK
